Set up Spotify

Create a free app at developer.spotify.com/dashboard, add a redirect URI that points back at this console, then set:

AR_SPOTIFY_CLIENT_ID=...
AR_SPOTIFY_CLIENT_SECRET=...
AR_SPOTIFY_REDIRECT_URI=http://<this-host>:8000/spotify/callback

The redirect URI must match the one in the Spotify app exactly. Restart, then reload this page and click Connect Spotify.

Playlist destinations

Select tracks in the console, then export them:

  • Plex — native playlist, or an M3U download of the in-library matches.
  • Spotify — native playlist, once connected above. Tracks are matched by title/artist search.
  • Tidal — supported via the unofficial API; not enabled in this build yet.
  • Apple Music — requires a paid Apple Developer account and browser-side MusicKit authorization; use the per-track Apple links or a transfer service (Soundiiz/TuneMyMusic) in the meantime.
Full reference for every setting is on the Help page and in audio_recognition/config.py.
